About the Encrypt and Decrypt Tool

This tool is designed to help you encrypt and decrypt messages using a combination of Caesar Cipher and Vigenère Cipher techniques. It provides a simple interface to input your text or numeric codes and perform the desired encryption or decryption operations.

How to Use

Encrypt: To encrypt a message, click the "Encrypt" button. Enter the text you want to encrypt in the provided input box and press "Submit". The tool will encrypt the text using a Vigenère Cipher with the keyword "ComeToTheCartelsCasino", convert the letters to numbers, and then apply a Caesar Cipher with a shift of 21. The encrypted message will be displayed in the output box.

Decrypt: To decrypt a message, click the "Decrypt" button. Enter the numeric code you want to decrypt in the provided input box and press "Submit". The tool will reverse the Caesar Cipher shift, convert the numbers back to letters, and then decrypt the text using the Vigenère Cipher with the same keyword. The decrypted message will be displayed in the output box.

Technical Details

The tool uses JavaScript to handle the encryption and decryption processes. The Vigenère Cipher is a method of encrypting alphabetic text by using a simple form of polyalphabetic substitution. The Caesar Cipher is a type of substitution cipher in which each letter in the plaintext is shifted a certain number of places down the alphabet.
